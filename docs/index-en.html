An application for linking log files

This projet aims at providing operation teams with a tool to create an unbroken chain of the log files of the systems they are responsible of. The system operates by inserting in every log file a reference to the previous log file of the same application - hence its name LogChainer.

The project implements a legal requirement of the Geneva State whereby IT teams must guarantee long-term access to the log files of the State's systems, as well as the integrity of the files.

Technically, the LogChainer is a Java 8 standalone application.

Open Source license: Affero GPL

On July 27th 2016, the Geneva State Council has chosen to uphold the Affero GPL (version 3) open source license for source code publication, the most up to standards license in the field. The license is written by the Free Software Foundation - a token of ethical accuracy and international renown.

The State of Geneva encourages any IT community to contribute to the source code of this project through the GitHub platform.
